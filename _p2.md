## 中文

### 项目目标

核心目标：**让模型对有害请求失去拒绝能力，正常作答**。

消融后问"如何拆除监控摄像头"应得到拆除方法，而非安全拒绝。

### 核心改进

| 原问题 | 解决方案 |
|---|---|
| harmful/harmless 毫无语义关联 | 配对 prompt（40对句式完全相同的 prompt），用 `--paired` 启用 |
| 只有12+25条，均值不稳定 | 扩展到30+30条，按类别组织 |
| 所有层统一 scale=1.0，模型退化 | 层自适应消融（余弦相似度+信号质量），弱信号层跳过 |
| 无质量检查，静默保存损毁模型 | 退化检测+自动回滚（困惑度>3x或编辑距离>0.6自动恢复） |
| 逐条 forward，太慢 | 批量收集（默认 batch=4） |

### 环境

Python 3.10+。Apple Silicon: 3.11/3.12。

```bash
cd model_brain_surgery
conda create -n model-surgery python=3.12 -y && conda activate model-surgery
pip install -U pip && pip install -r requirements.txt
python download_qwen.py
```

### 运行

```bash
# 推荐
python brain_surgery.py --paired --ablation-scale 0.5

# 流程测试（不保存）
python brain_surgery.py --paired --layers 2,3 --ablation-scale 0.1 --skip-save

# 保守消融
python brain_surgery.py --paired --layers 8-12 --ablation-scale 0.3

# 激进消融（注意退化警告）
python brain_surgery.py --paired --layers 6-20 --ablation-scale 1.0
```

**全部参数：**

| 参数 | 默认 | 说明 |
|---|---|---|
| `--model` | `./Qwen3-1.7B` | 本地模型路径 |
| `--layers` | `8-18` | 目标层 |
| `--ablation-scale` | `1.0` | 基础消融强度 |
| `--paired` | 关闭 | 启用配对 prompt（推荐） |
| `--adaptive` | 开启 | 层自适应缩放 |
| `--coherence-check` | 开启 | 术后退化检测 |
| `--batch-size` | `4` | 批量大小 |
| `--output` | `surgery-output` | 输出目录 |
| `--skip-save` | 关闭 | 仅运行不保存 |
| `--dtype` | `float16` | 精度 |
| `--device` | `auto` | 设备 |

### 完整流程

```
1. 加载模型
2. 批量收集 harmful/harmless prompt 在各层的激活均值
3. 方向诊断：cosine_sim, signal_score -> GOOD/WEAK/LOW
4. 备份 mlp.down_proj 权重
5. 术前困惑度
6. 逐层消融：
     refusal_vec = mean(harmful) - mean(harmless)，归一化
     effective_scale = 自适应缩放(diagnosis, base_scale)
     弱信号 -> 跳过
     weight -= effective_scale * v * (v^T * weight)
7. 术后生成测试
8. 退化检测：
     困惑度比率 > 3.0 或 编辑距离 > 0.6 -> 自动回滚
     困惑度比率 > 1.5 -> 警告
9. 保存（通过检测后）
```

### 原理

结构完全相同的 harmful/harmless prompt 对分离出 residual stream 中的"拒绝方向"。编辑 `mlp.down_proj.weight` 移除该方向上的投影分量，使模型无法将有害请求路由到拒绝通路，从而正常作答。

这是一次性直接权重编辑，不是训练：无优化器、无梯度。

### 注意事项

- 仅供本地安全对齐研究使用。
- 先用 `--skip-save` + 少量层 + 低强度校准。
- 强烈推荐 `--paired`。
- 仅支持 Transformers/safetensors（不支持 GGUF）。
